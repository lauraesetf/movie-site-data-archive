# movie-site-data-archive
该仓库用于建立影视网站基础数据档案，对互联网上可公开访问的影视类站点进行系统化信息整理。内容仅限站点层级的公开资料，不涉及影视内容本身。所有资料整理自公开网络环境，目的是形成可查询、可对照的站点数据集合。本项目不构成任何形式的推荐或宣传，仅作为信息整理与长期存档用途存在。
